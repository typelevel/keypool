keyedpool - A Keyed Pool for Scala

Quick Start

To use keyedpool in an existing SBT project with Scala 2.11 or a later version, add the following dependencies to your build.sbt depending on your needs:

libraryDependencies ++= Seq(
  "io.chrisdavenport" %% "keyedpool" % "<version>"
)
